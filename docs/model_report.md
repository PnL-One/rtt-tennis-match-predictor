# Model Report

## Цель

Модель предсказывает вероятность победы выбранного игрока в одиночном матче РТТ на основе информации, доступной до начала матча.

## Пользовательский прогноз

Для честной оценки качества модели используется фиксированный holdout test split. После выбора лучшего основного алгоритма интерактивный предиктор использует отдельную `production_main_model`: тот же выбранный тип модели переобучается на всей доступной истории матчей.

Это позволяет пользовательскому прогнозу учитывать самые свежие результаты, уже попавшие в датасет, не смешивая этот режим с расчетом test-метрик.

## Признаки

Основные группы признаков:

- официальный рейтинг и рейтинговые очки до матча;
- скорректированный рейтинг: очки на один зачетный турнир;
- относительные разницы рейтинговых показателей;
- ELO до текущего матча;
- ожидаемая вероятность победы по ELO;
- личные встречи до текущего матча;
- общий winrate игрока до текущего матча;
- давность прошлого матча и разница отдыха;
- опыт игрока;
- сила недавних соперников;
- признаки общих соперников;
- индикаторы отсутствующих рейтинговых данных.

Исторические признаки считаются в хронологическом порядке: текущий матч обновляет историю только после того, как признаки для него уже рассчитаны.

## Важные признаки

В последних прогонах наиболее сильными группами признаков были:

- относительная разница скорректированных рейтинговых очков;
- разница скорректированных рейтинговых очков;
- ожидаемая вероятность победы по ELO;
- разница рангов;
- разница ELO;
- относительная разница обычных рейтинговых очков.

Это ожидаемо: официальные рейтинги и ELO задают базовый уровень силы игрока, а дополнительные признаки помогают улучшить калибровку и качество на пограничных матчах.

## Ограничения

- Качество зависит от полноты и корректности парсинга страниц РТТ.
- ФИО игроков могут быть неоднозначными; такие случаи частично отфильтрованы и вынесены в audit-таблицы.
- Модель не учитывает внешние факторы: травмы, покрытие, поездки, текущую физическую форму вне истории матчей.
- Часть признаков в test может обновляться предыдущими матчами test-периода. Это соответствует сценарию последовательного прогнозирования по календарю, но не сценарию, где вся test-история полностью неизвестна заранее.
- Проект учебный и не предназначен для ставок или коммерческого прогнозирования.
