## Данные

Данные проекта получены из публично доступных источников Российского теннисного тура (РТТ) и использованы исключительно в учебных целях в рамках дипломного проекта по курсу «Машинное обучение в Python».

Проект не является коммерческим сервисом, не предназначен для ставок и не претендует на официальную аналитику РТТ.
